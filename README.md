# ⚡ IKS PRO

Sistema de gestión de registros — Web App

## Funcionalidades
- 🔐 Login con contraseña personal
- 📋 Registro de clientes (SAT FULL / IKS)
- 💸 Gastos e ingresos extra
- 🔍 Búsqueda y filtro por fecha
- 📊 Gráfica de ganancias mensuales
- 📅 Resumen mensual
- 📥 Exportar a Excel
- ☀️ Modo claro / oscuro
- 📱 Compatible con iPhone y Android

## Usar en iPhone
1. Abre Safari y ve a la URL de la app
2. Toca el botón compartir ⬆️
3. Selecciona **"Agregar a pantalla de inicio"**
4. Ya funciona como app nativa

## Datos
Los datos se guardan localmente en el navegador (IndexedDB).
